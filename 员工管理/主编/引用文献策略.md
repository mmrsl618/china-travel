## 一、分隔布局方案

### 问题
v1 方案用 1px 细线分隔正文与引用，线太细、留白不够，读者难以察觉"正文结束→参考资料开始"的分界。三哥反馈"太挤了"。

### v2 新方案（2026-06-23 重设计）

**HTML 结构模板：**（未变）

```html
<hr class="ref-separator">

<div class="article-refs">
  <p class="refs-note">本文基于 YYYY年MM月DD日 的政策信息。</p>
  <p class="refs-title">参考来源</p>
  <ul class="refs-list">
    <li>来源机构 — 文档/公告名称（更新日期）</li>
    <li>来源机构 — 文档/公告名称（更新日期）</li>
  </ul>
  <p class="refs-disclaimer">政策可能随时调整，递交申请前请核实最新信息。</p>
</div>
```

**CSS 关键值（已写入 `style.css`，作用域 `.article`）：**

| 元素 | class | v2 关键样式 | 与 v1 差异 |
|------|-------|-----------|-----------|
| 分隔线 | `.ref-separator` | `border-top: 2px solid #d0d0d0; margin: 3.5rem 0 2.5rem;` | 线加粗到 2px，上间距 2.5→3.5rem，下间距 1.75→2.5rem |
| 引用容器 | `.article-refs` | `border-left: 2px solid #e0e0e0; padding-left: 1rem;` | **新增**左侧灰色竖线，形成区块感 |
| 日期标注 | `.refs-note` | 间距 1rem→0.75rem | 微调 |
| 来源标题 | `.refs-title` | `font-size: 0.8125rem`（13px） | 字号 14→13px |
| 来源列表 | `.refs-list` | 间距 1rem→0.75rem | 微调 |
| 免责声明 | `.refs-disclaimer` | 无变化 | — |

**v2 设计意图：**
- 分隔线 2px + 上方 56px 留白 → 正文末尾有整段空白，线本身足够粗，读者不可能忽略
- 引用区左侧加 2px 灰色竖线 → 扫描页面时形成"侧边标记"，一眼认出是参考资料区块
- 字体保持 14px 灰色 → 不与正文争注意力

### 英文版模板

## 二、全站引用策略

### 核心原则：按文章类型分级

| 级别 | 文章类型 | 是否加引用 | 判断标准 |
|------|---------|-----------|---------|
| **必须加** | 政策/法规类（签证、过境免签、入境规定） | ✅ 必加 | 含官方政策声明、法规条款、费用数字 |
| **必须加** | 费用/数字类（签证费用、支付限额、汇率规则） | ✅ 必加 | 含具体数字且来源外部的数据 |
| **应该加** | 操作步骤类（支付开通、订票流程） | ✅ 建议加 | 涉及平台规则、系统行为，可能因版本更新变化 |
| **可加可不加** | 纯经验类（城市攻略、路线推荐、文化提示） | ⚠️ 可选 | 基于作者个人经验，无外部数据引用 |
| **不加** | 站点导航页、列表页、分类汇总页 | ❌ 不加 | 非文章详情页，无独立内容主体 |

### 触发条件速查

文章满足以下任一条件 → **必须**加引用文献：
- 引用了政府机构（NIA、外交部、领事馆）的公告或数据
- 给出了具体费用金额或数字上限
- 涉及法律法规条款或官方规定
- 引用了第三方平台（支付宝、微信、12306）的功能描述或限额

文章满足以下条件 → **可**加引用文献：
- 引用了行业报告或新闻来源的统计数据
- 涉及可能因版本更新而变化的功能指引

### 全站统一要求

- ✅ **格式统一**：所有带引用文献的文章，使用上述同一套 HTML 结构和 CSS 类
- ✅ **每条引用**包含：来源机构名称 + 文档/公告标题 + 日期
- ✅ **引用区始终**放在正文完结后、文章最末尾
- ✅ **不带链接**的纯文本引用 → 保持静态站简洁，读者可自行搜索
- ❌ **不在引用区**放营销链接、推广内容、或非来源性质的外部链接

## 四、后续文章规范

今后主编写新文章时：
1. 开写前用上表判断是否需要引用文献
2. 需要引用的，写作过程中记录每条数据/政策的来源
3. 正文完结后，按模板插入 `<hr class="ref-separator">` + 引用块
4. 中文版用中文标签（「参考来源」），英文版用英文标签（「Sources」）
