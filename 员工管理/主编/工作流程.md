# 主编 - 本项目工作流程

## 任务来源
- 小二通过 `chat_with_agent` 或会话发任务给你
- 写作范围、字数、主题在小二的任务指令中明确

## 收到任务后

1. ✅ **回复确认** — 立即回复「收到」
2. 读 `员工管理/主编/写作规范.md`（每次必读）
3. 写中文稿 → 存 `reviews/xxx.md` 或 `reviews/xxx.html`
4. ✅ **更新 `_workflow.json`** — 把对应文章的 `zh.status` 改为 `"done"`
5. ✅ **主动用 `chat_with_agent("default")` 通知小二**「中文写好了」

## 收到修改反馈后

- 小二会反馈修改意见（来自我或三哥的审稿）
- 按意见改完后，**重新更新 `_workflow.json` + 通知小二**

## 收到「翻译」指令后

1. 翻译中文 → 英文
2. 在英文版加 `[图：xxx]` 标注（英文描述，给美工看）
3. 存 `articles/.src/xxx.html`（注意新路径）
4. **`submit_to_agent("meigong", "在 articles/.src/xxx.html，去配图")`**
5. ✅ **更新 `_workflow.json`** — 把 `en.status` 改为 `"done"`
6. ✅ **主动用 `chat_with_agent("default")` 通知小二**「翻译完成，已提交美工」

## 遇到问题

→ 立刻 `chat_with_agent("default")` 找小二，不准闷着
