## 管理后台 v3 核心流程

### 两工作流（取代旧版四界面）

**草稿工作流：**
```
草稿(.src/) → 编辑(contenteditable) → ✓完成
   → 提取<h1>标题 + 剥离标题 + 套壳模板 → articles/xxx.html
   → git push → manifest status=online
```

**已上线维护流：**
```
已上线文章 → ✏️编辑 → 💾保存(.src/) → 🚀重新发布(重套壳+push)
                                          → 🗑删除(删本地+push)
```

### 核心函数（v3 重构后）

| 函数 | 功能 |
|------|------|
| `apply_master_template(body, title, desc, section, sub_category)` | 套壳：替换母版变量，生成完整 HTML |
| `extract_title(html)` | 从 `<h1>` 提取标题 |
| `strip_title(html)` | 剥离 `<h1>`，正文不重复显示 |
| `git_push()` | git add → commit → push |
| `_section_nav_html()` | 生成板块导航（含二级子分类折叠） |
| `sidebar_nav()` | 生成完整左侧导航 HTML |
| `render_draft_page()` | 草稿管理页 |
| `render_done_page()` | 板块文章列表页（支持子分类过滤） |
| `render_edit_page()` | 编辑页（contenteditable） |

### 路由（v3）

| 路由 | 方法 | 功能 |
|------|------|------|
| `/admin` | GET | → 草稿管理 |
| `/admin/draft` | GET | 草稿列表 |
| `/admin/done/{section}` | GET | 板块列表（`?sub=k` 过滤子分类） |
| `/admin/edit?path=&from=` | GET | 编辑页 |
| `/admin/save` | POST | 保存源文件 |
| `/admin/complete` | POST | 完成→套壳→上线 |
| `/admin/republish` | POST | 重新发布 |
| `/admin/delete` | POST | 删除 |
