# 图片规范 — china-travel-website

> **美工每次开工前第一件事必须读取此文件。**

## 配图来源（优先级顺序）

1. **搜图（首选）** — 去以下免费可商用网站搜真实素材：
   - **Unsplash** (`https://unsplash.com`) — 高质量真实摄影，CC0 协议，无需署名
   - **Pexels** (`https://www.pexels.com`) — 免费可商用，摄影/插画/视频截图
2. **截图** — 真实场景截图（如官网页面、APP界面），无版权风险
3. **生图（备选）** — 搜图/截图确实找不到合适素材时，允许自己生成

**配图时间控制：** 单篇文章总配图时间控制在 **5分钟内**，搜图最多花 5 分钟，找不到合适素材就直接生图，不准在这上面浪费时间。

## 基本原则

1. **严格按主编的描述配图**
2. **图片比例和尺寸：**
   - 📱 **手机界面截图** → 竖屏 **9:16**，`max-width: 400px`
   - 🖼️ **其他图片（场景/摄影等）** → 横屏 **16:9**，`max-width: 600px`
   - 所有图片 **居中**（`display: block; margin: 20px auto`）
3. **文件大小：** 单张 ≤ **200KB**，保证页面加载速度
3. **文字必须完整可读** — 无重叠、无遮挡、无截断
3. **文字对比度足够** — 不因颜色过浅而难以辨认
4. **布局整洁** — 同类元素对齐，互不干扰
5. **不加多余标题/标注**

## 审图标准

```
① 内容匹配主编描述
② 类型匹配（路线图/流程图/示意图）
③ 所有文字完整显示
④ 无文字重叠
⑤ 文字对比度足够
⑥ 布局整洁，互不干扰
⑦ 无多余/截断的标注
⑧ UI 穿帮检查 — 键盘/按钮等 UI 元素与背景/设备框体对齐，无脱离
⑨ 手机截图类 — 文字/内容不超出屏幕边界
```

## 生图质量专项检查（AI 生成图特别留意）

- ❌ **文字重叠** — 不同字段的文字不能互相覆盖
- ❌ **UI 穿帮** — 键盘按键与屏幕框体/设备实体对齐，不能悬空或错位
- ❌ **超框** — 手机屏幕截图/APP 界面类图片，文字不能超出屏幕边界
- ✅ **文字完整** — 每个中/英文字符完整显示，不截断、不遮挡
