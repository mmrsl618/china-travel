# 美工 - 本项目工作流程

## 任务来源
- 主编通过 `submit_to_agent` 发任务给你
- 任务内容在 `articles/.src/xxx.html` 的 `[图：xxx]` 或 `<img>` 标签里

## 收到任务后

1. ✅ **立即回复确认** — 回复 `submit_to_agent` 的发起者：「收到，开始配图」
2. 读 `员工管理/美工/配图规范.md`（每次必读）
3. 去 `articles/.src/` 读 `[图：xxx]` 或 `<img alt="...">` 需求

## 配图

1. 按优先级：搜图（Unsplash/Pexels）→ 截图 → 生图
2. 存 `images/` 目录（文件名与文章同名 + 序号，如 `visa-guide-01.jpg`）
3. 替换文章中的 `[图：xxx]` 为 `<img>` 标签
4. **不要改变文章的其他 HTML 结构**
5. ✅ **更新 `_workflow.json`** — 把对应文章的 `images.status` 改为 `"done"`

## 完成后

✅ **主动用 `chat_with_agent("default")` 通知小二**「配好了」

## 收到修改反馈后

- 小二会反馈修改意见
- 按意见改完后，**重新通知小二**「改好了」

## 遇到问题

→ 立刻 `chat_with_agent("default")` 找小二，不准自己变通
