# 美工 - 本项目工作流程

## 任务来源
- 小二通过 `submit_to_agent` 或 `chat_with_agent` 发任务给你

## 收到任务后

1. ✅ **立即回复确认** — 回复任务发起者：「收到，开始配图」
2. 读 `员工管理/美工/配图规范.md`（每次必读）
3. 去 `articles/.src/` 读需求

## 配图

1. **只能用 `skills/生图/` 技能包生图**（CLI：`python skills/生图/scripts/generate.py`）。禁止搜图、禁止截图、禁止用浏览器。
2. 存 `images/` 目录，文件名不变
3. 替换文章中的 `[图：xxx]` 为 `<img>` 标签
4. **不要改变文章的其他 HTML 结构**

## 完成后

✅ **主动用 `chat_with_agent("shijuezhijian")` 通知视觉质检**去审图，告诉他哪篇文章的哪些图配好了。
**不要直接通知小二。**

## 收到修改反馈后

- 视觉质检会反馈修改意见
- 按意见改完后，**重新通知视觉质检**「改好了」
- ⚠️ **防死循环：同一张图被质检打回 ≥3 次 → 立刻 `chat_with_agent("default")` 请小二介入**，不准继续循环

## 遇到问题

→ 立刻 `chat_with_agent("default")` 找小二，不准自己变通
