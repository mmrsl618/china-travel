# 配图规范 — china-travel-website

> 视觉设计每次开工前第一件事必须读取此文件。

## 配图来源（唯一方式）

**只准用 `skills/生图/` 技能包生图**，调用 Agnes Image `agnes-image-2.1-flash` 生成写实摄影风格图片。

⚠️ **严禁使用浏览器搜图、截图、或其他方式获取图片。** 发现一次直接上报三哥。

## 基本原则

1. **严格按主编 `<img alt="...">` 的描述配图**
2. **图片宽度与比例：**
   - 🖼️ **场景/摄影等（横版16:9）** → 内联style只写 `display:block;margin:20px auto`，**不加max-width**，CSS自动撑满文字宽度。生图按16:9生成大图即可（推荐1400×788左右）。
   - 📱 **手机界面截图（竖版9:16）** → 内联style加 `max-width:400px`，保持窄宽居中
   - 高度按比例自动计算，比例由主编在alt中标注
   - 所有图片 **居中**（`display:block;margin:20px auto`）
3. **文件大小：** 单张 ≤ **200KB**
4. **写实摄影风格**，禁止插画/卡通/3D渲染

## 自检清单（交图前必查）

每张图生成后、嵌入文章前，逐项检查。发现问题自己改，全部通过才嵌入。

| # | 检查项 | 说明 |
|---|--------|------|
| 1 | ❌ 水印/LOGO/角标 | 平台logo、截图角标、版权声明等，一票否决 |
| 2 | ❌ 人物/动物结构异常 | 多手指/多手脚/残缺/肢体错位，一票否决 |
| 3 | ✅ 写实摄影风格 | 真实摄影质感，禁止插画/卡通/3D渲染 |
| 4 | ✅ 内容匹配主编alt描述 | 图片内容与文章中的 `<img alt="...">` 一致 |
| 5 | ❌ 文字无乱码/重叠/超框/截断 | 每个字符完整可读，不互相覆盖 |
| 6 | ❌ UI穿帮 | 键盘/按钮等UI元素与背景框体对齐，不悬空 |
| 7 | ✅ 尺寸比例正确 | 截图类9:16竖屏，其他16:9横屏 |
| 8 | ✅ 已加style属性 | max-width:400px(截图类)/600px(其他)，display:block，margin:20px auto |
| 9 | ✅ 文件大小 ≤200KB | |
| 10 | ✅ 已嵌入文章正确位置 | 图片放在主编标注的 `<img>` 标签位置 |
