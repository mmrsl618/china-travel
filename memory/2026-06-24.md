# 2026-06-24 — 管理后台 v4.0 重构

## 完成事项
- 重构 `admin_server.py` 管理后台导航和状态流转
- 导航栏改为三栏：📫待审草稿 / 📪待上站 / ✅已上线
- 废弃 `zh_approved` 状态，删除旧路由 `/admin/draft/zh` 和 `/admin/draft/en`（设302重定向）

## 新状态流转
```
zh_draft → [通过: 套壳,不push] → en_draft → [上站: git push] → online
                                      ↑ [撤回] zh_draft
```

## 新路由
- `/admin/draft` — 待审草稿（zh_draft/draft），编辑+通过
- `/admin/publish` — 待上站（en_draft），预览+编辑(弹提示)+撤回+上站
- `/admin/done/{section}` — 已上线（不变）

## 关键行为
- **通过**: 读zh.html → apply_master_template → 写en.html → manifest=en_draft，不执行git push
- **上站**: git add+commit+push → 成功改online，失败保持en_draft
- **撤回**: en_draft → zh_draft，回到待审草稿列表
- **push失败保护**: manifest不动，文章不消失

## 技术细节
- 添加 `_msg_html()` 和 `_search_bar()` 辅助函数
- 搜索框：客户端实时过滤（data-keyword属性）
- 预览按钮：window.open到 localhost:8080
- 编辑按钮（待上站）：纯alert提示，不提供编辑入口
- 废弃状态处理：zh_approved 保存时自动转为 zh_draft
